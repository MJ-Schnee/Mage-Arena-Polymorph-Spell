## Installation

1. **Prerequisites**:
   - BepInEx 5.4.21
   - ModSync
   - BlackMagicAPI

2. **Installation**:
- After downloading the mod, place the files in the following locations:

   ```
   MageArena/
   └── BepInEx/
       └── plugins/
           └── PolymorphSpell/
			   └── PolymorphSpell.dll
   ```
